Proyecto 2

Volver

Asiento lúdico infantil, a partir de materia prima creada de textiles desechados. Esta propuesta inicia debido al vertedero de ropa que se encuentra en el norte de Chile, pudiendo hacerse utensilios para diferentes áreas. Para este proyecto se escogieron todo tipo de ropa para personas esepctuando denim, debido a que esta tela es rigida. Posteriormente por cada prenda se crearon ovillos para poder empezar a tejerlo. Se parametrizo a través de una tabla , en la que se hicieron 68 puntos xx 41 corridos, ademas se utilizaron 4 hebras de trabajo. Para su relleno se utilizo partes de las prendas que no se usaron, como puños, cuellos, etiquetas, etc., además de una capa de nada para que este quede aireado y no con sensación de dureza, al ser para niños y de uso colaborativo, debiese cumplir ciertas normas.
